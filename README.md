# Registre des Volontaires - CLTCRH

Application web pour gérer le registre des volontaires du Comité Local de Tabarre de la Croix-Rouge Haïtienne (CLTCRH).

## Fonctionnalités

### ✨ Gestion des Volontaires
- **Ajout** de nouveaux volontaires avec toutes leurs informations
- **Modification** des informations existantes
- **Suppression** de volontaires
- **Recherche** rapide par nom, prénom, téléphone, zone ou compétences

### 📊 Statistiques
- Total des volontaires
- Volontaires actifs/inactifs
- Répartition par sexe
- Top 5 des zones

### 📁 Export
- Exportation des données au format CSV (compatible Excel)

### 💾 Stockage
- Stockage local (localStorage) - fonctionne hors ligne
- Pas besoin de base de données ou serveur

## Couleurs

L'application utilise les couleurs officielles de la Croix-Rouge :
- **Rouge** (#CC0000) - Primaire
- **Blanc** (#FFFFFF) - Fond
- **Noir** (#000000) - Textes et accents

## Installation

1. Clonez ou téléchargez ce dépôt
2. Ouvrez le fichier `index.html` dans votre navigateur

Aucune installation n'est requise - l'application fonctionne directement dans le navigateur!

## Données collectées

Pour chaque volontaire :
- Nom et Prénom
- Sexe
- Date de naissance
- Téléphone
- Email
- Adresse
- Zone/Tabarre
- Compétences
- Disponibilité
- Statut (Actif/Inactif)
- Date d'inscription

## Utilisation

### Ajouter un volontaire
1. Cliquez sur le bouton "Nouveau Volontaire"
2. Remplissez le formulaire
3. Cliquez "Enregistrer"

### Modifier un volontaire
1. Cliquez sur l'icône 📝 dans la colonne "Actions"
2. Modifiez les informations
3. Cliquez "Enregistrer"

### Supprimer un volontaire
1. Cliquez sur l'icône 🗑️ dans la colonne "Actions"
2. Confirmer la suppression

### Rechercher
Utilisez la barre de recherche pour filtrer les volontaires par nom, prénom, téléphone, zone ou compétences.

### Exporter les données
Cliquez sur "Exporter CSV" pour télécharger toutes les données au format Excel/CSV.

## Compatibilité

- Compatible avec tous les navigateurs modernes
- Responsive (adapté aux mobiles et tablettes)
